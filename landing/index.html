REDDIT API USE

ChaDongCha uses the Reddit API in read-only mode to power its community identification feature. We surface publicly posted photos from r/whatisthiscar as a guessing mechanic for players.

We do not collect Reddit account data, store credentials, or post on behalf of users. All Reddit content shown in the app includes attribution to the original poster.

HOW WE USE THE API

  • Read public posts and top comments from r/whatisthiscar (read-only)
  • Display post images in-app with attribution to the Reddit OP
  • Use top-voted comments as answer data for the identification mechanic
  • No user login, no OAuth, no writes to Reddit
  • Rate-limited to well within Reddit API guidelines
